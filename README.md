# Financial Tracker v3.9 Multi-Project

Replace the existing `index.html` in the FTracker GitHub repository with this version.

## v3.9 enhancements

- Weekly Actuals cells crossing a calendar-month boundary are shown with a diagonal split.
- Weekly values are allocated to each month using Monday-Friday working-day proportions.
- Weekly totals use the same split-cell presentation.
- A monthly reconciliation summary is shown beneath the weekly grid.
- The Monthly Reconciliation interval provides a read-only monthly pivot for Finance and invoice checking.

The underlying source of truth remains the weekly actual value. Month allocations are calculated for display and reconciliation only.
